Read the README.md file within this skill, and explore the other available files (tokens/, components/, assets/, guidelines/, and "Guia Visual Novo Ego.html").

The aesthetic in one line: warm near-black backgrounds + a single champagne-gold metallic accent + high-contrast Didone serif (authority) paired with a soft serif/italic (humanity) and a neutral grotesk (UI). Grayscale photography, generous whitespace, slow/calm motion, no emoji.

If creating visual artifacts (landing pages, social art, slides, mocks, throwaway prototypes, etc), copy assets out (`assets/monogram.svg`, `assets/wordmark.svg`) and create static HTML files that link `styles.css` for tokens, or inline the token values. If working on production code, copy assets and read the rules here to become an expert in designing with this brand.

If the user invokes this skill without any other guidance, ask them what they want to build or design, ask a few questions (audience, single page vs system, how many variations), and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.

Hard rules: one gilt CTA per view; text is `--sand-50` not pure white; gold is an accent never a fill; photos are black & white; no emoji, no bounce, no rainbow gradients.
