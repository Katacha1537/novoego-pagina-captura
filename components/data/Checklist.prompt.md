Gold-tick deliverable list; use 2 columns for longer sets.

```jsx
<Checklist columns={2} items={[
  'Design responsivo e otimizado para conversão',
  'Copywriting persuasivo e focado em resultados',
  '100% pronta para rodar',
  'Acompanhamento para ajustes de conversão',
]} />
```

Or compose rows directly with `<ChecklistItem>`.
