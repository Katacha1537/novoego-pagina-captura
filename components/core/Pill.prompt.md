Eyebrow capsule that flags a section above a headline; 1–4 tracked uppercase words.

```jsx
<Pill icon={<Sparkles size={14} />} tone="gold">A Nova Ciência da Mente</Pill>
<Pill icon={<Users size={14} />}>Equipe Especializada</Pill>
```

Tones: `default` (neutral hairline) and `gold` (warm tint + gold border). Always sits directly above an `h1`/`h2`, never inline in body copy.
